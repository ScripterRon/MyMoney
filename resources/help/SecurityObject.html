Securities

A security describes an investment and is the basis for all transactions in an investment account. Each security has a price history which is used to value the security when generating reports and calculating capital gains. The security price can be updated manually or obtained automatically over the internet.

Security Types

  • Bond Mutual Fund

  • CD

  • Corporate Bond

  • Municipal Bond

  • Stock

  • Stock Mutual Fund

  • Treasury Note

Security Fields

  • Security Type – The security type is used when generating the pie chart for the investment account positions tab.

  • Security Name – Identifies the security in transaction dialogs and reports.

  • Ticker Symbol – The ticker symbol is used when updating security prices over the internet. Securities without a ticker symbol will not be updated.

  • Security Hidden – Hidden securities will not be displayed in the transaction dialog selection fields. Transactions with hidden securities will still be displayed.
