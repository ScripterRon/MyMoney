Overview

This is a personal money management program. It is a general-use program with the exception that the estimated tax function is set up for a single person living in New York.

The following account classes are supported:

  • Asset accounts

  • Bank accounts

  • Credit accounts

  • Investment accounts

  • Loan accounts

The following security classes are supported for use with investment accounts:

  • Certificate of Deposit (CD)

  • Corporate bond

  • Municipal bond

  • Bond mutual fund

  • Stock

  • Stock mutual fund

  • Treasury Note

The following reports can be generated:

  • Bond amortization/accretion sorted by account and date

  • Capital gains sorted by sale date

  • Stock holdings sorted by account and security

  • Transactions sorted by date, account, name or category

The following calculators are provided:

  • Bond amortization/accretion

  • Compound interest with optional annual withdrawals

  • Estimated taxes

  • Bond yield-to-maturity

Income and expense transactions can be classified by category (Medical, Automobile, Utility, Computer Game, etc). New categories can be created at any time. Cross-account transactions are supported (for example, paying a credit card bill from a checking account or transferring money from a checking account to a savings account)

A net worth graph can be generated for any period of time. Stock prices can be automatically updated through an internet connection (Yahoo! is used to retrieve the current stock quotes).

To get started, you need to create at least one account. Once you have created an account, select it from the View drop-down menu. You can now start creating transactions within the account (deposit money, withdraw money, transfer money between accounts, etc).
