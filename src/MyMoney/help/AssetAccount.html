Asset Account

An asset account is used to track the value of an asset such as a home, car or boat. An asset transaction reflects an increase or decrease in the value of the asset. For example, the value of a home might increase as a result of a new property assessment. Or the value of a car might decrease as a result of depreciation. The current asset value is included in the net worth calculations for the Overview panel and the Net Worth graph.

Account Functions

  • New Transaction – Create a new asset transaction.

  • Edit Transaction – Modify an existing asset transaction.

  • Delete Transaction – Delete an existing asset transaction.
