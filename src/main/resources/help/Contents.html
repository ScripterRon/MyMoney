Help Contents

Click on an underlined link for more information about the associated topic.

  • Overview – MyMoney overview

  • File – Database operations

    • Open – Close the current database and open another database. You will be prompted to save any pending changes to the current database before it is closed.

    • Save – Save changes made to the current database.

    • Exit – Exit the program. You will be prompted to save any pending changes to the database before the program ends.

  • View – Display non-hidden accounts

    • Asset Accounts – Asset account processing.

    • Bank Accounts – Bank account processing.

    • Credit Accounts – Credit account processing.

    • Investment Accounts – Investment account processing.

    • Loan Accounts – Loan account processing.

  • Objects – Create/Edit/Delete program objects

    • Accounts – Create, edit and delete accounts.

    • Categories – Create, edit and delete categories.

    • Schedules – Create, edit and delete schedules.

    • Securities – Create, edit and delete securities.

  • Calculators

    • Amortization – Calculate bond amortization and accretion using the yield-to-maturity.

    • Compound Interest – Calculate compound interest with optional annual withdrawals.

    • Yield to Maturity – Calculate yield-to-maturity for a bond purchased at a discount or premium.

  • Graphs

    • Net Worth – Display a net worth graph. The graph will not be accurate for years with archived transactions or securities with incomplete price histories.

  • Reports

    • Amortization – Generate a bond amortization/accretion report.

    • Capital Gains – Generate an investment capital gains report.

    • Investments – Generate an investment portfolio report.

    • Transactions – Generate a transaction report. The transactions can be sorted by date, account, name or category.

  • Online

    • Update Prices – Update stock prices using an internet connection.
